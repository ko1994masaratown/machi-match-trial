# Machi Match

「助けてほしい町」と「力を活かしたい人」をつなぐ、国民参加型まちづくりプラットフォームです。

全国の市町村が抱える人手不足・高齢化・産業衰退・空き家増加などの課題をSOSスコアで可視化し、移住・副業・単発アルバイト・ボランティア・リモート支援など、多様な関わり方でマッチングします。

## 主な機能

- SOSマップ：自治体カード一覧、SOSスコア色分け、検索・期間フィルタ
- 市町村詳細モーダル：課題／強み／関わり方／イベント／声／返礼品の6タブ
- 関わり方：単発アルバイト、短期滞在、ボランティア、副業、リモート支援、移住、外国人材、企業支援
- GPS連携：現在地取得、距離表示、近隣イベント・自治体の優先表示
- イベント一覧：花火、祭り、文化祭、スポーツ、朝市など
- AI推薦：スキル・希望・出身地をもとに自治体を3件推薦
- マイページ：お気に入り、活動履歴、返礼品記録
- デモ用ログイン切替：右上ボタンでゲスト／ログイン表示を切替

## Vercelへのデプロイ手順

1. GitHubにこのプロジェクトをpush
2. VercelでGitHubリポジトリを選択
3. Framework Preset は Vite を選択
4. Build Command は `npm run build`
5. Output Directory は `dist`
6. Deploy

## Claude API連携について

この完成版では、APIキー漏洩を防ぐため、ブラウザからClaude APIを直接呼びません。

環境変数 `VITE_CLAUDE_PROXY_URL` を設定した場合のみ、任意のバックエンド／APIプロキシに `POST { prompt }` を送信します。未設定の場合は、デモ用の自然なモックAI回答で動作します。
